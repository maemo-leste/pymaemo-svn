A widget which allows the user to enter a telephone number.

Description

A Single Line Editor widget which allows user to enter a telephone number. There are two modes to choose from (coerce/free format). Coerce format has three fields which are placed horizontally. Fields are country code, area code and number. When nothing is inputted to the fields assisting text is displayed in a field. Tapping on the field highlights the field and allows user to input numbers.

The widget is used where a user should enter a phone number. Free format should be used wherever possible as it enables the user to enter the phone number in the format he likes in addition to entering DTMF strings as a part of the phone number. The format can not be changed at runtime.

Constructor

Creates a new Telephone Editor.

hildon.TelephoneEditor(format)
format Format of the editor. See the table below.
Returns A new hildon.TelephoneEditor.

 

Telephone Editor Formats
String Format
"free" Free format.
"coerce" Coerce format, three fields.
"coerce-country" Coerce format, only country field.
"coerce-area" Coerce format, country and area fields.

Methods

set_editable

Specify whether the fields in coerse format are editable or not. This function is ignored if the editor is in free mode. The number or area and number parameters are ignored if the editor is in "coerce-area" or "coerce-country" formats, respectively.

hildon.TelephoneEditor.set_editable(country, area, number)
country Set whether (True or False) the country field in coarce format should be editable or not.
area Set whether (True or False) the area field in coarce format should be editable or not.
number Set whether (True or False) the number field in coarce format should be editable or not.

get_show_plus

Gets the visibility status of the plus sign in front of coerce format's country field.

hildon.TelephoneEditor.get_show_plus()
Returns True if the plus sign is being shown and False otherwise.

set_show_plus

Show or hide the plus sign in front of coerce format's country field.

hildon.TelephoneEditor.set_show_plus(show)
show True to show the plus sign and False otherwise.

get_show_border

Get the visibility status of the border around the widget.

hildon.TelephoneEditor.get_show_border()
Returns True if the border is being shown and False otherwise.

set_show_border

Show or hide the border around the widget.

hildon.TelephoneEditor.set_show_border(show)
show True to show the border and False otherwise.

set_widths

Set widths of the fields in coerse format. Country and area parameters are ignored, if the editor is in free mode. The number or area and number parameters are ignored if the editor is in "coerce-area" or "coerce-country" mode, respectively.

hildon.TelephoneEditor.set_widths(country, area, number)
country Width (characters) of the country field in coarce mode.
area Width (characters) of the area field in coarce mode.
number Width (characters) of the number field in coarce mode.

get_country

Get text in the country field in coarse format. This method must not be called if in free mode.

hildon.TelephoneEditor.get_country()
Returns The text in the country field.

set_country

Set text in the country field in coarse format. This method must not be called if in free mode.

hildon.TelephoneEditor.set_country(country)
country Text to be in the country field in coarse format.

get_area

Get text in the area field in coarse format. This function must not be called if in free mode or in HILDON_TELEPHONE_EDITOR_FORMAT_COERCE_COUNTRY mode.

hildon.TelephoneEditor.get_area()
Returns The text in the area field.

set_area

Set text in the area field in coarse format. This method must not be called if in free mode or in "coerce-country" mode.

hildon.TelephoneEditor.set_area(area)
area Text to be set in area field.

get_number

Get text in the number field in all formats. If in free mode this method returns the whole phone number and if in coarce mode only the number field's is returned. This method must not be called if the editor is in "coerce-area" or "coerce-country" modes.

hildon.TelephoneEditor.get_number()
Returns Text in the number field.

set_number

Set text in the number field in all formats. If in free mode this method sets the whole phone number and if in coerce mode only the number field's is set. This method must not be called if the editor is in "coerce-area" or "coerce-country" modes.

hildon.TelephoneEditor.set_number(number)
number Text to be set in number field.
