What is Python for Maemo?

Python for Maemo (PyMaemo for short) main objective is to make possible to use Python programming language as the scripting and development language for Maemo Platform, providing a better alternative for fast prototyping and programming in Maemo environment besides the C programming language.

But what is Maemo? Maemo is a Linux based development platform from Nokia for compliant handheld devices like Nokia 770 Internet tablet and Nokia N800 Internet tablet. The platform is composed of popular free/open source software components, which are widely deployed in the leading Linux desktop distributions today. Maemo adapts these components to the handheld environment and contributes new technologies.

Why we choose Python? First of all forget about C development because it takes too much time to develop and debug.

Python is for serious programming and to have fun. Python has a nice syntax, it's easy to learn and powerful enough for a vast range of applications, this is why we choose Python for Maemo.

Components

The current PyMaemo version is derived from Python 2.5 with customizations in order to fit constrains of space (memory/storage) and speed required by small devices. There are bindings for Python to write games with Pygame (SDL), to write graphical applications with GTK/Hildon, numerical calculus, Internet protocols HTTP, SMTP, POP, IMAP, XML processing and so on.

From the latest release of Python for Maemo you can use:

  • Python 2.5.1
  • Numerical Python 23.8
  • PyGObject 2.12.2
  • PyGTK 2.10.3
  • Python Hildon 0.8.7
  • Python GStreamer 0.10.5
  • Python Bluez 0.9.1
  • Python D-Bus 0.71
  • Python Osso 0.2
  • Python XML 0.8.4
  • Pygame 1.7.1
  • Pyrex 0.9.4.1
  • PyId3lib 0.5.1
  • Python-Conic 0.1 (Only Bora)
  • Python-Imaging 1.1.6 (libjpeg)
  • Runtime 0.4-10
  • iPython 0.7.3

This set of packages is also known as: Python for Maemo 2.2/3.0.

The old Python for Maemo distributions (1.0, 2.0 and 2.1) is now deprecated.
