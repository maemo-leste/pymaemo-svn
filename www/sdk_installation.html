SDK Installation (scratchbox)

The first thing is to add the maemo extras repository (where Python is located) to your list of package sources. To do that you'll have to edit the file /etc/apt/sources.list, which should look like this: 

deb http://repository.maemo.org/ bora free non-free
deb-src http://repository.maemo.org/ bora free
deb file:/home/[username]/maemo-sdk-nokia-binaries_3.1 bora explicit

Add the following line to it:

deb http://repository.maemo.org/extras bora free non-free

Now resynchronize your package index files with the following command:

[sbox-SDK_X86: ~] > apt-get update

After that you are ready to install python2.5 plus bindings. All Python 2.5 for Maemo packages are named python2.5-*, so you can install all at once with apt-get if you want to doing the following: 

[sbox-SDK_X86: ~] > apt-get install python2.5*

To see all python-related packages issue this command

[sbox-SDK_X86: ~] > apt-get cache search python*

Note that you'll see also scratchbox's own python (version 2.3) which you must not use for your maemo development.
