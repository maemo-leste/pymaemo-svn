Installation (OS2007 users)

Warning: It is necessary to remove ALL applications that depends on Python2.4 BEFORE you remove python2.4-runtime. This is necessary to ensure that all python2.4 packages will be correctly removed.

To perform automatic installation, use both files listed below. First Base repository and then Python runtime.

Base repository
Python 2.5 installer
Download size: 5.6 MB
Space used after installation: around 17 MB

For manual installation instructions and installing development packages on the device, click here.

If you want to install Python on maemo SDK (scratchbox), check the steps here.
