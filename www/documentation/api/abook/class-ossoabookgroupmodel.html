Description

The abook.GroupModel is a model that lists user-defined groups (see abook.Group). Additional groups can be added using the abook.GroupModel.add_group() method.

    def remove_group(group)

Removes group from model. Do not remove abook.ManualGroup instances from the model, if they have been automatically added. Use abook.ManualGroupManager to manage manual groups.
