abook Constants

abook Constants — the built-in constants of the abook module

Synopsis

Account Type Constants
Group Model Constants
Presence Status Constants

Description

Account Type Constants

Bitmask that specifies what type of features accounts should have.

abook.ACCOUNT_TYPE_CHAT = 1>>0Account that supports chat
abook.ACCOUNT_TYPE_VOIP = 1>>1Account that supports VoIP
abook.ACCOUNT_TYPE_EMAIL = 1>>2Account that supports email
abook.ACCOUNT_TYPE_CHAT = 0x111Account that supports everything

Presence Status Constants

The different statuses a presence can have.

abook.PRENSENCE_ONLINE = 0Online
abook.PRENSENCE_AWAY = 1Away
abook.PRENSENCE_OFFLINE = 2Offline
abook.PRENSENCE_PENDING = 3Pending
abook.PRENSENCE_UNKNOWN = 4Unknow

Group Model Constants

The data columns of the model. Data cell data can be retreived using gtk.TreeModel.get()

abook.GROUP_MODEL_COL_GROUP=0The column containing abook.Group objects.
