abook.ManualGroup — Manually created groups.

Synopsis

class abook.ManualGroup(abook.Group):

Ancestry

+-- GObject
  +-- abook.Group
    +-- abook.ManualGroup

Description

A manually created group contents are specified by the user. The object however will only match regular (i.e. non-blocked) contacts.

Manual groups are managed by abook.ManualGroupManager
