abook.Presence.get_status

    def get_status()
Returns :a Presence Status

Retreives the status of this instance. Note that the "status" property can be monitored by connecting to the gobject.GObject "notify" signal with "status" as detail, i.e. "notify::status".

abook.Presence.get_icon_name

    def get_icon_name()
Returns :The icon name currently associated with this instance.

Retrieves the icon name associated with the current status of this instance. The icon name can be used to look up the appropriate icon to represent this instance until the "status" property changes.

abook.Presence.compare

    def compare(b)
presence :a abook.Presence
Returns :a int

Compares the status of this presence to the status of presence. See Presence Status for the weights assigned to different statuses.
