"sort-weight"ReadSort weight of the group. A lower value results in a higher sorting priority. Default value: 0.

The "refilter-contact" abook.Group Signal

    def callback(group, contact, user_param1, ...)
group:the group that received the signal
contact:the evolution.ebook.EContact to be refiltered.
user_param1:the first user parameter (if any) specified with the connect() method
...:additional user parameters (if any)

The refilter-contact signal is emitted when contact has to be refiltered by any containing filters. This is used when contact's membership to the group has changed, for example.
