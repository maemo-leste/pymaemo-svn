Description

The abook.AlphaBar is a bar of locale-specific shortcuts to ranges of alphanumeric characters. When a shortcut is activated, the cursor of the associated abook.TreeView will be moved to the first row starting with one of the characters from the range.

There are two ways to use an abook.AlphaBar. The first one is to merge it into a gtk.UIManager using abook.AlphaBar.merge(). The second one is to create an independent array of buttons using abook.AlphaBar.create_button_row().
